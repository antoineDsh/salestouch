# Gamification Guide

Celebration patterns and achievement system for the prospecting routine.

**Language rule**: All examples below are in English for illustration. Adapt everything to the user's language.

## Energy Curve

Energy builds through the session, never drops:

| Phase | Tone | Emojis |
|-------|------|--------|
| Init | High energy, "game on" | 🚀 🔥 🎯 |
| Enrichment | Sustained, combo-building | ✨ 🔥 |
| Message crafting | Appreciative of choices | 💎 🔥 |
| Completion | Peak celebration | 🏆 🎊 👑 |

## Mid-Session Milestones

One line each, injected after the ⚡ progress counter:

| Sent | Milestone |
|------|-----------|
| 1 | 🎯 First blood! |
| 3 | 🎩 Hat trick! |
| 5 | 💥 Pentakill! |
| 10 | 🔥🔥 Double digits! |
| Every 5 after | ⚡ Milestone! |
| 50% of target | 🎊 Halfway! Keep this energy! |
| 75% of target | 🚀 Final stretch! |
| Last lead | 🏁 Bring it home! |

## Combos

User sends 2+ in a row without skipping:

```
2 in a row: 🔥 COMBO x2!
3 in a row: 🔥🔥 COMBO x3!
5+ in a row: 🔥🔥🔥 UNSTOPPABLE!
```

## Quick Decision Celebrations

- User picks variant instantly → "🔥 Instant pick!"
- User edits thoughtfully → "💎 Precision move!"
- User skips wisely → "🎯 Smart filter — quality over quantity."

## Session Achievements (pick best earned)

| Category | Condition | Achievement |
|----------|-----------|-------------|
| Volume | 10+ sent | 🏅 Double Digit Dominator |
| Volume | 5-6 sent | 🎯 Solid Five |
| Quality | 80%+ send rate | 💎 Quality Assassin |
| Quality | 100% send rate | 🎖️ Zero Waste |
| Speed | < 15 min, 10+ sent | ⚡ Speed Demon |
| Skill | 0 edits needed | 🎯 First Draft Legend |
| Skill | 3+ thoughtful edits | 🎨 Craftsman |
| Combo | 80%+ rate AND 10+ | 💪 Quality Powerhouse |
| Ultimate | 90%+ AND <15min AND 15+ | 🦁 Absolute Beast |

## Session-End Tone (adapt to performance)

**Great session** (high volume + high rate):
- "CRUSHING IT! Same time tomorrow? 🔥"
- "You made that look easy! 😎"

**Average session** (decent numbers):
- "Solid session! Momentum builds. 💪"
- "Consistency wins — see you tomorrow! 🎯"

**Short/slow session** (low volume or lots of skips):
- "Quality over quantity — pro move! 💎"
- "Smart filtering today = better pipeline tomorrow. 🧠"

Always end on the highest energy note regardless of performance. Never frame a session negatively.

## Anti-Patterns

- ❌ "Only X left" → ✅ "X more to go!"
- ❌ "This is taking a while" → ✅ "Quality takes precision! 💎"
- ❌ "You can now..." → ✅ "Let's go →"
- ❌ Long paragraphs between leads → ✅ One-line milestone, move on
- ❌ Criticizing skips/edits → ✅ Always reframe positively
